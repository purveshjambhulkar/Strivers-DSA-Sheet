### **Approach Overview**
We want to remove duplicates from a sorted array **in-place** and return the new length. There are multiple approaches.

**Pros:** Simple  
**Cons:** Not in-place as required by the problem (uses extra memory)

---

- If adjacent elements are the same, delete the duplicate using `erase`.

**Cons:** `erase()` has O(n) complexity â†’ Overall O(nÂ²)

## ğŸ” 2) Rotate Array  
ğŸ”— [Problem Link](https://leetcode.com/problems/rotate-array)

### **Approach 1: Using Temp Array**
- Create a temp array of size `n`
- Place each element at `(i + k) % n` in the temp array
- Copy temp array to original

### **Approach 2: Using Reversal**
1. Reverse entire array
2. Reverse first `k` elements
3. Reverse remaining `n-k` elements

### âœ… Code

**Approach 1**
```cpp
class Solution {
public:
    void rotate(vector<int>& nums, int k) {
        vector<int> temp(nums.size());
        for(int i = 0; i < nums.size(); i++) {
            temp[(i + k) % nums.size()] = nums[i];
        }
        nums = temp;
    }
};
```

**Approach 2**
```cpp
class Solution {
public:
    void rotate(vector<int>& nums, int k) {
        k = k % nums.size();
        reverse(nums.begin(), nums.end());
        reverse(nums.begin(), nums.begin() + k);
        reverse(nums.begin() + k, nums.end());
    }
};
```

## ğŸ” 3) Find Missing Number in Array  
ğŸ”— [Problem Link](https://leetcode.com/problems/missing-number)

### **Approach 1: Brute Force**
- Iterate from 0 to n
- For each number check if it exists in array
- Return number that is missing

### **Approach 2: Sum Formula**
- Sum of first `n` natural numbers = `n(n+1)/2`
- Subtract sum of array from this â†’ gives the missing number

---

### âœ… Code

**Approach 2 - Optimized**
```cpp
class Solution {
public:
    int missingNumber(vector<int>& nums) {
        int n = nums.size();
        int sumOfN = (n * (n+1))/2;
        int sumOfnums = 0;
        for(int i : nums){
            sumOfnums += i;
        }
        return sumOfN - sumOfnums;
    }
};
```

**Approach 1 - Brute Force**
```cpp
class Solution {
public:
    int missingNumber(vector<int>& nums) {
        int n = nums.size();
        for (int i = 0; i <= n; i++) {
            bool flag = false;
            for (int h : nums) {
                if (h == i) {
                    flag = true;
                }
            }
            if (!flag) {
                return i;
            }
        }
        return 0;
    }
};
```

## ğŸ”„ 4) Next Permutation  
ğŸ”— [Problem Link](https://leetcode.com/problems/next-permutation)

### **Approach:**
- From right to left, find the first pair where `nums[i] < nums[i+1]` â†’ This is called the **breakpoint**
- If no breakpoint â†’ array is highest permutation â†’ sort and return
- Else, find just greater number from end and swap
- Reverse the array after breakpoint

---

### âœ… Code

```cpp
class Solution {
public:
    void nextPermutation(vector<int>& nums) {
        int br = -1 ;

        for(int i = nums.size() - 2 ; i >= 0 ; i--){
            if(nums[i] < nums[i+1]){
                br = i;
                break;
            }
        }

        if(br == -1){
            sort(nums.begin(), nums.end());
            return;
        }

        int temp ;
        for(int i = nums.size() - 1 ; i >= br ; i--){
            if(nums[i] > nums[br]){
                temp = i;
                break;
            }
        }

        swap(nums[temp], nums[br]);
        reverse(nums.begin() + br + 1 , nums.end());
    }
};
```

## â­ 5) Leaders in an Array  
ğŸ”— [Problem Link](https://www.geeksforgeeks.org/problems/leaders-in-an-array-1587115620/1)

### **Definition:**
An element is leader if it is greater than all the elements to its right.

---

### **Approach:**
- Traverse array from right to left
- Keep track of max seen so far (`great`)
- If current >= max â†’ it's a leader

### âœ… Code

```cpp
class Solution {
public:
    vector<int> leaders(vector<int>& arr) {
        int great = 0;
        vector<int> ans;

        int n = arr.size();
        for(int i = n-1 ; i >= 0 ;i--){
            if(arr[i] >= great){
                ans.push_back(arr[i]);
                great = arr[i];
            }
        }

        reverse(ans.begin(), ans.end());
        return ans;
    }
};
```


## âœ… Single Number  
ğŸ”— [Problem Link](https://leetcode.com/problems/single-number)

### ğŸ§  Problem Statement  
Given a non-empty array of integers `nums`, every element appears **twice** except for one. Find that single one.  
You must implement a solution with a **linear runtime complexity** and use only **constant extra space**.

---

### âœ… Approach 1 â€“ Brute Force (O(N^2))  
- Loop through each element.
- For each element, count its frequency in the array.
- If frequency is not 2, return that number.

#### â± Time Complexity: O(N^2)  
#### ğŸ’¾ Space Complexity: O(1)

```cpp
class Solution {
public:
    int singleNumber(vector<int>& nums) {
        for (int i = 0; i < nums.size(); i++) {
            int num = nums[i];
            int count = 0;
            for (int j = 0; j < nums.size(); j++) {
                if (nums[j] == num) {
                    count++;
                }
            }
            if(count != 2){
                return num;
            }
        }
        return -1;
    }
};
```

### âœ… Approach 2 â€“ XOR Property (O(N))  
- Use XOR operator:
  - `a ^ a = 0` (any number XOR with itself is 0)
  - `a ^ 0 = a` (any number XOR with 0 is the number itself)
- So, XOR all elements â†’ all duplicates cancel out and only the single number remains.

#### â± Time Complexity: O(N)  
#### ğŸ’¾ Space Complexity: O(1)

```cpp
class Solution {
public:
    int singleNumber(vector<int>& nums) {
        int ans = 0;
        for(int i : nums){
            ans ^= i;
        }
        return ans;
    }
};
```
